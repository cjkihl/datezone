## `Calendar`

Represents a wall clock time with individual date and time components.
This type is used for representing time as it appears on a wall clock
in a specific timeZone, without any timeZone offset information.

```ts
import { Calendar } from "@datezone/calendar";
```

## `calendarToTimestamp`

Converts a wall-clock `Calendar` value in a specific time-zone to a UTC timestamp (milliseconds since epoch).

```ts
import { calendarToTimestamp } from "@datezone/calendar";
```

<CodeExample file="reference/calendar/calendarToTimestamp.ts" title="calendarToTimestamp" />

## `timestampToCalendar`

Converts a timestamp in UTC to a calendar in a specific timeZone.

```ts
import { timestampToCalendar } from "@datezone/calendar";
```
