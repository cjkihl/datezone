## wallTimeToTS

Converts zoned time (local time in a specific timezone) to UTC timestamp.

```typescript
function wallTimeToTS(
  year: number,
  month: number, // 1-based (1 = January)
  day: number,
  hour: number,
  minute: number,
  second: number,
  ms: number,
  timeZone?: TimeZone
): number
```
