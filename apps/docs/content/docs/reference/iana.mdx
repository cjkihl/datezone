## Complete Timezone List

The `TimeZone` type includes all IANA timezone identifiers. Here are some examples by region:

### Africa (50+ timezones)
- `Africa/Abidjan`, `Africa/Accra`, `Africa/Cairo`, `Africa/Casablanca`, `Africa/Johannesburg`, `Africa/Lagos`, `Africa/Nairobi`, etc.

### America (150+ timezones)  
- `America/New_York`, `America/Chicago`, `America/Denver`, `America/Los_Angeles`, `America/Toronto`, `America/Mexico_City`, `America/Sao_Paulo`, etc.

### Asia (100+ timezones)
- `Asia/Tokyo`, `Asia/Shanghai`, `Asia/Kolkata`, `Asia/Dubai`, `Asia/Singapore`, `Asia/Seoul`, `Asia/Bangkok`, etc.

### Europe (60+ timezones)
- `Europe/London`, `Europe/Paris`, `Europe/Berlin`, `Europe/Rome`, `Europe/Madrid`, `Europe/Amsterdam`, `Europe/Stockholm`, etc.

### Pacific (30+ timezones)
- `Pacific/Auckland`, `Pacific/Fiji`, `Pacific/Honolulu`, `Pacific/Guam`, `Pacific/Tahiti`, etc.

### Australia (10+ timezones)
- `Australia/Sydney`, `Australia/Melbourne`, `Australia/Perth`, `Australia/Adelaide`, `Australia/Darwin`, etc.

### Special Zones (50+ timezones)
- `UTC`, `GMT`, `Etc/UTC`, `Etc/GMT+1` through `Etc/GMT-14`, etc.
