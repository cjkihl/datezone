# Month Utilities (`datezone/month`)

This page documents all exported functions from the `month` module in `datezone`.

## Functions

### startOfMonth

Returns the start of the month (first day, 00:00:00.000) in the given timezone.

```ts
startOfMonth(ts: YearMonthOptions | number, timeZone?: TimeZone): number
```
