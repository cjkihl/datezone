# Day Utilities (`@/datezone/day`)

This page documents all exported functions from the `day` module in `@/datezone`.

## Functions

### addDays

Adds a number of days to a timestamp or options object.

```ts
addDays(ts: DayOptions | number, days: number, timeZone?: TimeZone): number
```

<Tabs items={["Local", "Timezone"]}>
  <Tab value="Local">

```ts title="Local time"
addDays(Date.now(), 5)
```
  </Tab>
  <Tab value="Timezone">

```ts title="With timezone"
addDays(Date.now(), 5, "America/New_York")
```
  </Tab>
</Tabs>

### weekDayName

Returns the localized weekday name.

```ts

```

<Tabs items={["en-US", "fr-FR"]}>
  <Tab value="en-US">

```ts title="en-US"

```
  </Tab>
  <Tab value="fr-FR">

```ts title="fr-FR"

```
  </Tab>
</Tabs>
