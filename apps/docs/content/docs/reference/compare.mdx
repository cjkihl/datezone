# Compare Utilities

Compare utilities provide functions for comparing dates and timestamps, including relative time comparisons and date equality checks.

## Daily Comparisons

Check if dates are today, tomorrow, or yesterday.

```typescript
function isToday(date: OptionsOrTimestamp, timeZone: TimeZone): boolean
function isTomorrow(date: OptionsOrTimestamp, timeZone: TimeZone): boolean
function isYesterday(date: OptionsOrTimestamp, timeZone: TimeZone): boolean
```

## Time-based Comparisons

Check if dates are in the past or future.

```typescript
function isPast(date: OptionsOrTimestamp, timeZone: TimeZone): boolean
function isFuture(date: OptionsOrTimestamp, timeZone: TimeZone): boolean
```
