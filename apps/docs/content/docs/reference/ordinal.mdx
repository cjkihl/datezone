## Supported Locales

The `formatOrdinal` function supports the following locales with their specific ordinal formatting rules:

### English (en)
- **1st, 21st, 31st, etc.**: Numbers ending in 1 (except 11) use "st"
- **2nd, 22nd, 32nd, etc.**: Numbers ending in 2 (except 12) use "nd"  
- **3rd, 23rd, 33rd, etc.**: Numbers ending in 3 (except 13) use "rd"
- **4th, 5th, 6th, etc.**: All other numbers use "th"
- **11th, 12th, 13th**: Special cases that use "th"

### Spanish (es)
- **All numbers**: Use "º" suffix (masculine ordinal indicator)

### French (fr)
- **1st**: Uses "er" (premier)
- **All others**: Use "e" (deuxième, troisième, etc.)

### German (de)
- **All numbers**: Use "." (period) suffix

### Italian (it)
- **All numbers**: Use "º" suffix

### Portuguese (pt)
- **All numbers**: Use "º" suffix (masculine form)

### Dutch (nl)
- **All numbers**: Use "e" suffix
