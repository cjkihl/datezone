## Install with Bun, npm, or yarn

```package-install
datezone
```

## Requirements
- Node.js 18+, Deno or Bun
- Works in modern browsers and serverless environments

## Usage Example

```ts
import { parse, format, toTimezone } from "datezone";

const date = parse("2024-06-01T12:00:00Z");
const formatted = format(date, "yyyy-MM-dd HH:mm zzz");
const local = toTimezone(date, "America/New_York");
```

See the [Basic Usage](/docs/basic-usage) guide for more details.
