# Datezone API Reference

This section documents all public functions, types, and options available in the Datezone library.

## Core Functions
- `parse(input: string | Date): Date` — Parse a date string or Date object
- `format(date: Date, format: string): string` — Format a date with tokens
- `toTimezone(date: Date, tz: string): Date` — Convert a date to a specific IANA timezone
- `compare(a: Date | string, b: Date | string): -1 | 0 | 1` — Compare two dates
- `addDays(date: Date, amount: number): Date` — Add days to a date
- ...and more

## Types
- `DatezoneDate` — Internal date representation
- `Timezone` — IANA timezone string

> See the source code and README for full details and advanced usage.
