Datezone makes working with dates, times, and timezones simple and reliable.

## Installation

Install with Bun, npm, or yarn:

```bash
bun add datezone
# or
npm install datezone
# or
yarn add datezone
```

## Importing

```ts
import { parse, format, toTimezone, compare, addDays } from "datezone";
```

## Parsing Dates

```ts
import { parse } from "datezone";

const date = parse("2024-06-01T12:00:00Z");
```

## Formatting Dates

```ts
import { format } from "datezone";

const formatted = format(date, "yyyy-MM-dd HH:mm zzz");
// => "2024-06-01 12:00 UTC"
```

## Timezone Conversion

```ts
import { toTimezone } from "datezone";

const ny = toTimezone(date, "America/New_York");
```

## Comparing Dates

```ts
import { compare } from "datezone";

compare("2024-06-01", "2024-06-02"); // -1
compare("2024-06-01", "2024-06-01"); // 0
compare("2024-06-02", "2024-06-01"); // 1
```

## Adding/Subtracting Dates

```ts
import { addDays } from "datezone";

const tomorrow = addDays(date, 1);
```
