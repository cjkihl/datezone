Datezone is a lightweight, high-performance date and timezone utility library for JavaScript and TypeScript. It provides a modern, ergonomic API for parsing, formatting, comparing, and manipulating dates and times, with full support for IANA timezones and DST.

JavaScript's native `Date` object has no timezone support, and existing libraries require additional plugins like `@date-fns/tz` or `date-fns-tz` that add complexity with wrapper objects like `TZDate`.

Datezone is built from the ground up with timezones natively integrated into every function, eliminating the need for plugins or special wrapper objects.
